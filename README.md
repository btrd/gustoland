# Requirement
PostgreSQL

Ruby 2.4.0 & bundler

Pour mac
```
brew install rbenv
rbenv install 2.4.0
gem install bundler
```


# Installation
```
bundle
bundle exec rails db:setup
bundle exec rails server
```
